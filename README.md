## Discourse Announcement Bar

A Discourse theme component to display a dismissible announcement bar.

The bar text and button label are localized via `locales/en.yml`. Changing the text automatically re-shows the bar for all users who previously dismissed it.

### Plugin outlet

The `plugin_outlet` setting controls where the bar renders. Default is `above-site-header`. Other useful outlets:

- `above-site-header` — above the full site header
- `below-site-header` — below the header, above content
- `above-main-container` — inside the main content area, adds bottom margin
